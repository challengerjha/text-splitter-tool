# 📝 Text Splitter Tool

A simple and efficient web-based tool to split a large block of text into smaller chunks based on a specified word count. Perfect for content writers, editors, students, and developers who want to segment large content into manageable pieces.

## 🚀 Features

- Paste or write a long article or paragraph.
- Specify how many words you want per chunk.
- Automatically splits the content.
- Each chunk has a handy **Copy** button.

## 🔧 How It Works

1. Paste your content in the text box.
2. Enter the desired number of words per chunk (e.g., 120).
3. Click "Split Text" to view the output.
4. Use the "Copy" button next to any chunk to copy it instantly.
